#Command usage :

/addtodo todoItem
Adds todo Item to database for particular channel.

/marktodo todoitem
Deletes todoitem if exist

/listtodos 
List all todoItem for particular channel

/edittodo previousName/New Name       #### "/ " is used as separator
Renames the todoitem
